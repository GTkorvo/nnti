How to use WebSolver:

  1. Step 1 is the definition of the linear system matrix, the solution vector, the right-hand side vector, and possibly an initial vector. After step 1, you have one or more "ProblemID", which identify the linear problem you are solving.
  2. Step 2 defines the parameters for the linear algebra object analysis, direct and iterative solutions;
  3. Step 3 performs all the computatons;
  4. Step 4 collects the results.

NOX Compiling Guidelines

Supported platforms

  • Linux (primary development platform)
  • SGI (ASCI Blue-mountain)
  • Solaris
  • ASCI Red
  • IBM w/KCC compiler (ASCI White)

General Comments

  • Trilinos and MPI: While NOX does not explicitly use parallel communication, to compile and run the example and test suites, Trilinos and MPI must be installed on the machine. For parallel versions run configure with the option --enable-mpi.

Linux

    % configure --enable-mpi \
            TRILINOS_HOME=/sierra/Stable/Trilinos/2.0b \
            TRILINOS_ARCH=TFLOP \
            TRILINOS_COMM=MPI 

% make SUBDIRS=src NOX_AR="ar rcv"

     

SGI

    
% configure CC=cc CXX=CC CFLAGS=-g CXXFLAGS="-g -DSGI64 -LANG:std" \
            --enable-mpi --with-mpilib=-lmpi \
            TRILINOS_HOME=/sierra/Stable/Trilinos/2.0b \
            TRILINOS_ARCH=SGI64 TRILINOS_COMM=MPI

% make SUBDIRS=src NOX_AR="ar rcv"

     

Solaris

    
% configure CC=/sierra/Stable/MPICH/1.2.1/bin/mpicc \
            CXX=/sierra/Stable/MPICH/1.2.1/bin/mpiCC \
            --enable-mpi CFLAGS=-g CXXFLAGS=-g \
            TRILINOS_HOME=/sierra/Stable/Trilinos/2.0b \
            TRILINOS_ARCH=SOLARIS TRILINOS_COMM=MPI

% make SUBDIRS=src NOX_AR="CC -xar -o"

     

ASCI Red (TFLOP)

  • Cross-compiling for janus.sandia.gov on solaris: 
    % configure CC=cicc CXX=ciCC --enable-mpi CFLAGS=-g \
            CXXFLAGS="-g -DTFLOP -I${TFLOPS_XDEV}/tflops/include/CC/RW" \
            --with-mpilib=-lmpi \
            TRILINOS_HOME=/sierra/Stable/Trilinos/2.0b \
            TRILINOS_ARCH=TFLOP TRILINOS_COMM=MPI --host=i686

% make SUBDIRS=src NOX_AR="ar rcv"

     

IBM w/KCC

    % configure CC=mpcc CXX=mpKCC CFLAGS=-g CXXFLAGS=-g \
            --enable-mpi TRILINOS_HOME=/sierra/Stable/Trilinos/2.0b \
            TRILINOS_ARCH=IBMSP TRILINOS_COMM=MPI --host=powerpc

% make SUBDIRS=src NOX_AR="KCC -o"

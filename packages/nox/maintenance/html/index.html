Mailing Lists

We have three new mailings lists:

  • NOX-Developers - This mailing list is for our "official" discussions regarding the NOX project. All the emails to this list will be archived. That should help with conforming to the new SQE practices.
     
  • NOX-CVS - This is the mailing list to which the cvs commit messages will be sent. This is NOT archived.
     
  • NOX-Users - This is a mailing list for our future user base. It can be used for the discussion of user-related issues such as compiling on different platforms, requests for modifications, and so on. This list is also archived.

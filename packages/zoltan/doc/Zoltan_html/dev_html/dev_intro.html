Introduction and General Principles

The goal of the Zoltan project is to design a general-purpose tool for dynamic load balancing.  This tool will be a suite of load-balancing algorithms to be used by a variety of applications.  It will, thus, be used by many application developers and be added to by many algorithm developers.  Software projects of this scale need general guidelines and principles so that the code produced is easily maintained and added to.  We have tried to keep restrictions on developers to a minimum.  However, we do require that a few coding practices be followed.  These guidelines are described in the following sections:

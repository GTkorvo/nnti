Compilation

The Zoltan distribution includes a main (top-level) Makefile with compilation instructions for several target architectures. When the library is compiled for a specific target architecture, A, a subdirectory, Obj_A, is created, and Makefile_sub is copied into that directory. This Makefile is then called with architecture-specific values passed in from the main Makefile. Currently supported architectures and their targets are listed in the table below.
 
Target Architecture
smos Intel Paragon running SUNMOS (sipapu).
tflop Sandia/Intel Teraflops computer (janus).
sun Sun workstations running Solaris.
sunos Sun workstations running SunOS4.
cplant CPlant parallel computing system (alaska).
nistsun Sun workstations at NIST running Solaris.
nistlinux Intel PC workstations at NIST running Linux.
Targets and architectures currently supported in the Makefiles of the Zoltan distribution.
 

The top-level Makefile includes compiler, include-file, and library paths that are specific to Sandia's MPCRL LAN (building 980); these paths must be adjusted to compile the library in different environments.

The GNU gmake utility is used to build dependency files and compile the library for a given target:

gmake [options] <target>
where the options that can be specified are listed below.
Options to gmake:  
    NO_PARMETIS=1 Do not compile and link with ParMETIS in the Zoltan library.  If this option is specified, no ParMETIS libraries are needed when the application is linking.  Otherwise, a ParMETIS library path must be specified to build the application.
    YES_FORTRAN=1 Include Fortran support in the Zoltan library.  By default, the Zoltan library is built without the interface that allows use from Fortran applications.  If this option is specified, then the Fortran interface is compiled and included in the library.  Use of this option requires that a Fortran 90 (or 95, or later) compiler is available.
After compilation, the library libzoltan.a is stored in the appropriate Obj_* directory for the specified architecture.

FORTRAN: Compiling Zoltan

Existing implementations

The main Makefile has not yet been updated to include the Fortran option in the process of building the Zoltan library. So far, the Fortran interface has been built and tested only through fdriver, and only on NIST machines. This section will be written after the Makefile has been updated. The main issues to be discussed will be using YES_FORTRAN=1 in the make command, and mention of lb_user_const*.h with a pointer to the section below. There will be some other parameters (selection of which Fortran compiler on Linux, and selection between LAM and MPICH), but these will probably affect only the NIST builds so I'm not sure they're worth mentioning.

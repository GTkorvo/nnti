FORTRAN: Compiling Zoltan

Existing implementations

To include the Fortran interface in the Zoltan library, use the YES_FORTRAN parameter in the make statement, for example
gmake YES_FORTRAN=1 tflop
Before compiling the library, make sure that the application's lb_user_const.h and lb_user_const.f90 have been placed in lb/ and fort/ repectively.

Currently, only the NIST targets support the Fortran option. The Sandia targets should be added soon.

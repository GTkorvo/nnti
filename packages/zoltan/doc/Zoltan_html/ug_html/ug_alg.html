Algorithms

The following dynamic load-balancing algorithms are currently included in the Zoltan library:
Recursive Coordinate Bisection (RCB)
ParMETIS
Octree Partitioning (OCTPART)
The input string argument for LB_Set_Method is listed for each method. The parameters listed can be specified for the method through the parameters array in the argument list of LB_Set_Method. For further analysis and discussion of the algorithms, see [Hendrickson and Devine].

Initialization Example

Before computing new decompositions using Zoltan, an application must initialize the load-balancing library. This initialization is accomplished through calls to LB_Initialize and LB_Create_Object. In the figure below, an example of the initialization calling sequence is shown. In this example, LB_Initialize is called using the argc and argv arguments to the main program. Then a load-balancing object is returned by the call to LB_Create_Object. In this example, all processors will be involved in load balancing, as MPI_COMM_WORLD is passed as the communicator to be used by the load-balancing routines.
 
/* Initialize the LB library */ 
double lb_params[LB_MAX_PARAMS_SIZE]; 
struct LB_Struct *lb; 
float version; 
... 
LB_Initialize(argc, argv, &version); 
LB_Initialize_Params_Array(&(lb_params[0])); 
lb = LB_Create_Object(MPI_COMM_WORLD);
Typical calling sequence for initialization of the Zoltan library.
